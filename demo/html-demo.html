HTML 规范

1. DOCTYPE 位置

<DOCTYPE html > 必须在文档的最开始，避免引起文档的怪异渲染模式。

2. 移动端的 meta 值

<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">
<meta content="email=no" name="format-detection" />
<meta name="format-detection" content="telephone=no" />

3. PC 端 IE edge

<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1"> 。

4. 标签闭合

基础要求。

5. 不使用 inline 样式。

禁止在页面的head通过 <style> 定义样式，和在 dom 中通过使用 style 的方式定义 inline 样式。

6. 4个空格缩进

使用4个空格代替制表符缩进。

7. 注释规则

模块通过 <-- 模块名字 -- > 开始，<-- /模块名字 -- > 结束，状态切换注释在DOM切换处注释标注。
